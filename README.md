# HTML

## Introduction

### HTML Elements

HTML Element คือส่วนประกอบต่าง ๆ ของเว็บ ส่วนประกอบคือ แท็กเปิด เนื้อหา แท็กปิด เช่น

```html
<h1>Hello World</h1>
<p>This is paragraph</p>
```

### HTML Attribute 

HTML Attribute คือ คุณสมบัติ/ลักษณะ มันจะให้ข้อมูลเพิ่มเติมเกี่ยวกับองค์ประกอ
- HTML Elementทั้งหมดสามารถมีAttributeได้
- Attributeจะถูกระบุที่แท็กเริ่มต้นเสมอ

ตัวอย่าง
```html
<p id="SixSeven">Hello</p>
<a href="https://www.google.com">Google Link</a>
```
id href คือ attribute ที่อยู่ด้านในtag

## Basic

### Heading
- Heading หรือ ส่วนหัว คือชื่อเรื่องหรือคำบรรยายที่คุณต้องการแสดงบนหน้าเว็บ
- Headingจะมีอยู่6ระดับ
- h1จะใหญ่สุด เรียงลำดับลงมา h6จะเล็กที่สุด

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```
