Box-Sizing

คุณสมบัติ (Property) ที่ใช้ในการกำหนดวิธีคำนวนขนาดความกว้าง (Width) และความสูง (Height) ของ Element ว่าจะให้รวมเอาขนาดของ เส้นขอบ (Border) และ ระยะห่างภายใน (Padding) เข้าไปด้วยหรือไม่

Border-Box

ไม่ว่าเราจะใส่ Padding หรือ Border หนาเท่าไหร่ก็ตาม ขนาดรวมภายนอกของกล่องจะไม่มีวันเกินค่า Width และ Height ที่เรากำหนดไว้
